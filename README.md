Hand-Gesture-Controlled-Wheel-Chair

Overview

This project presents a smart wheelchair system that we can controll using hand gestures. It is designed to assist physically challenged individuals by providing an intuitive and user-friendly way to navigate without relying on traditional joysticks or manual controls.

The system detects hand movements using sensors and translates them into motion commands such as forward, backward, left, and right.

Features

 1.Gesture-based control for easy operation
 2.Wireless communication between control unit and wheelchair
 3.Real-time response to hand movements
 4.Compact and low-cost design
 5.User-friendly interface

Working Principle

The system uses an accelerometer or gesture sensor mounted on the user's hand to detect motion in different directions.

1. Hand tilted forward → Wheelchair moves forward
2. Hand tilted backward → Wheelchair moves backward
3. Hand tilted left/right → Wheelchair turns accordingly

The sensor data is processed by a microcontroller and transmitted wirelessly to the wheelchair control unit. The control unit then drives the motors through a motor driver circuit.
